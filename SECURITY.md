1. **JANGAN** mempublikasikan temuan Anda di chat publik (Discord/Minecraft) atau di Github Issues publik.
2. Laporkan temuan Anda secara privat melalui **Tiket Tiketing / DM Admin** di server Discord resmi kami, atau hubungi Developer utama kami **Ran Dev** langsung melalui kontak resmi WhatsApp berikut:
